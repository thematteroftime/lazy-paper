# Internal Documentation

This directory contains historical and internal-development documents for `lazy-paper`. These are not part of the public-facing project documentation but are retained for development continuity.

- **HANDOFF.md** — Original development hand-off / status log from the initial implementation (May 2026). Useful for understanding empirical findings and design decisions, but not authoritative for current behavior; see the top-level README and `docs/ARCHITECTURE.md` instead.
- **superpowers/** — Planning and specification documents from the initial development session.

External developers should consult the top-level `README.md`, `docs/ARCHITECTURE.md`, and `CONTRIBUTING.md` instead.
